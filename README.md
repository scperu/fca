# Fernando Clemente Arana - Campaña Acción Popular

Sitio web oficial de la campaña de Fernando Clemente Arana para la provincia de Huancavelica.

## 🚀 Tecnologías

- **HTML5** - Estructura semántica
- **CSS3** - Variables CSS, Flexbox, Grid
- **JavaScript (ES6+)** - Web Components, Intersection Observer
- **Web Components** - Componentes reutilizables nativos
- **Font Awesome** - Iconos
- **Google Fonts** - Tipografías premium

## 🏗️ Arquitectura

### Web Components
El sitio utiliza **Web Components nativos** para crear componentes reutilizables y encapsulados:

- `<fca-header>` - Header con navegación
- `<fca-hero>` - Sección hero
- `<fca-stats>` - Estadísticas
- `<fca-pilares>` - Pilares de transformación
- `<fca-conoceme>` - Sección "Conóceme"
- `<fca-huancavelica>` - Sección Huancavelica
- `<fca-noticias>` - Noticias
- `<fca-sumate>` - Formulario de registro
- `<fca-footer>` - Footer

### Ventajas
- ✅ **Reutilización** - Los componentes se usan en múltiples páginas
- ✅ **Encapsulamiento** - Shadow DOM para estilos aislados
- ✅ **Sin dependencias** - No requiere librerías externas
- ✅ **Escalable** - Fácil agregar nuevos componentes
- ✅ **SEO amigable** - Renderizado en el servidor
